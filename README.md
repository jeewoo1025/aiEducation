## 성능 측정 방법
1. **BLEU** 
* Bilingual Evaluation Understudy
* 기계번역의 성능이 얼마나 뛰어난가를 측정하기 위해 사용함
* 기계 번역 결과와 사람이 직접 번역한 결과가 얼마나 유사한지 비교하여 번역에 대한 성능을 측정하는 방법
* 높을 수록 성능이 더 좋다
* 장점 : 언어에 구애받지 않음, 계산 속도가 빠름
